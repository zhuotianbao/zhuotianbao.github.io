css垂直居中

发表于 2017-02-17 | | 本文总阅读量 次

有时候我们要让一个DIV垂直居中，有时该DIV的宽高都是不确定的，有时宽高是确定的，那么我们如何让它垂直居中呢？接下来我们将要从传统方式，css3，以及最新的flex布局等方法去达到我们想要的效果。

.box{
width:200px;
height:200px;
background-color:red;
position:relative;
}
.center{
background-color:green;
top:50%;
left:50%;
position:absolute;
transform:translate(-50%,-50%);
}

.box{
width:200px;
height:200px;
background-color:red;
display: flex;
justify-content:center; //水平方向居中
align-items: center; //垂直方向居中
}
.center{
background-color:green;
}

.box{
width:200px;
height:200px;
background-color:red;
display: table-cell;
vertical-align:middle;
text-align:center;
}
.center{
background-color:green;
display:inline-block;
}

.box{
width:200px;
height:200px;
background-color:red;
position:relative;
}
.center{
width:50px;
height:50px;
background-color:green;
position: absolute;
margin:auto;
left:0;
top:0;
right:0;
bottom:0;
}

.box{
width:200px;
height:200px;
background-color:red;
position:relative;
}
.center{
width:50px;
height:50px;
background-color:green;
position: absolute;
margin:auto;
left:50%;
top:50%;
margin-left:-25px;
margin-top:-25px;
}
```
